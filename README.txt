ASK JORDAN CORE v1.5.0
======================

تحديث حقيقي مبني فوق v1.4.0.

الجديد:
- Search V2: فهم سنة الموديل مثل "كامري 2018".
- دمج سنة الموديل مع السعر والمحافظة والتصنيف في نفس البحث.
- ترتيب النتائج ومنع ظهور إعلان لا يطابق سنة الموديل المطلوبة.
- عداد مشاهدات لكل إعلان.
- عداد نقرات واتساب والاتصال والمشاركة.
- لوحة أداء داخل حساب البائع.
- إظهار إحصاءات كل إعلان داخل قائمة "إعلاناتي".
- إظهار عدد المشاهدات داخل تفاصيل الإعلان.

مهم:
الإحصاءات في هذا الإصدار محفوظة محليًا على الجهاز ولا تحتاج تشغيل SQL جديد.
ارفع محتويات هذا المجلد إلى جذر مستودع GitHub ثم Commit changes.
