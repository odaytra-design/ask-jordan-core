Ask Jordan Core v1.7.0 — GitHub Ready

التحديثات الفعلية:
- لوحة مشرف تظهر فقط للحساب الذي role=admin.
- إحصاءات: الإعلانات النشطة والمحذوفة والمستخدمون والبلاغات المفتوحة.
- مراجعة جميع الإعلانات وتغيير حالتها إلى نشط أو محذوف أو مباع.
- نظام بلاغات من صفحة تفاصيل الإعلان.
- مراجعة البلاغات وقبولها أو رفضها.
- قائمة قراءة للمستخدمين مع تمييز المشرفين.
- الاحتفاظ بكل ميزات v1.6.0 السابقة.

طريقة النشر:
1) ارفع جميع ملفات هذا المجلد إلى جذر GitHub واستبدل الملفات القديمة.
2) افتح Supabase > SQL Editor.
3) شغّل upgrade-v1.7.0.sql مرة واحدة.
4) عدّل آخر سطر في ملف SQL وضع رقم حسابك بدل 07XXXXXXXX ثم شغّله لتعيين حسابك كمشرف.
5) سجل خروج ثم دخول، وستظهر لوحة المشرف أعلى الموقع.
